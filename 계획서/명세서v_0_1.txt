1.클라이언트는 ID와 PW를 치고 확인버튼을 누를시 자신의 ID,PW,PORT번호를 해당 서버의 IP,port번호로 전송한다.

2.서버는 전송받은 ID,PW를 확인후 ID,PW가 맞다면 해당 이용자에게 로그인 됐다는 메시지를 전송 ID,PW가 틀릴 경우 틀렸다는 메시지를 해당 IP,PORT번호로 로그인 불가 메시지를 전송한다.

3.클라이언트가 로그인되면 자신이 메시지를 보낼 상대방의 접속 확인 후 메시지를 작성하여 전송을 누른다.

4. 서버는 해당 메시지를 상대방 id의 ip,port번호로 전송한다.
 

●일정시간동안 사용자가 메시지 전송, 수신이 없다면 해당 사용자가 이용 중인지 확인 메시지를 클라이언트에게 전송하며 만일 클라이언트에게 답이 없다면 로그아웃 된 것으로 가정한다.
 
●클라이언트가 로그아웃 시 서버에 로그아웃 메시지를 전송하고 서버는 해당 사용자가 로그아웃이 됐다고 처리한다.

●로그아웃 시 서버는 상대방 ip와 port번호를 삭제한다.


예외: 둘이 동시에 메시지를 서버가 동시에 수신하는 경우는 없다.
